# Jogo_Bug-Hunter

Jogo simples de plataforma, projetado para atender aos requisitos da disciplina de Técnicas de Programação do curso de Engenharia de Computação da UTFPR (Universidade Tecnológica Federal do Paraná).

# Objetivos a cumprir:

* Makefile (para rodar em qualquer PC)
* Comportamento inimigos
* Movimento de personagem (MARCOS)
* Algoritmos de colisoes
* Projeteis
* Obstáculos
* Fases

# Ideias:

* O personagem jogável será um aracnídeo e terá poder de projetar teias para capturar suas presas.
* A primeira fase será uma fase do estilo móvel horizontal.
* A segunda fase será no estilo móvel vertical. Haverá uma corrente de ar vertical que possibilitará o personagem 
a voar usando um planador de teia com base em energia estática, bem como algumas espécies reais de aranhas.
